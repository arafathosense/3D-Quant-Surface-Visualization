# 3D Quant Surface Visualization

This project generates animated 3D financial visualizations inspired by quant research. It uses Python, numpy, and matplotlib to create interactive, colorful 3D scatter plots with principal axes, suitable for exploring synthetic or real financial data.

<img width="987" height="687" alt="1" src="https://github.com/user-attachments/assets/8ab3d124-ed23-467e-96c8-b3c56a029600" />

## What is this project about?
- Visualizes high-dimensional financial data in 3D
- Animates the view and color for dynamic exploration
- Demonstrates quant research-style graphics (e.g., PCA, risk surfaces)
- Easily extensible for your own data or models

## How to fork and run
1. **Fork this repository** on GitHub (top right corner: "Fork")
2. **Clone your fork**:
   ```bash
   git clone https://github.com/arafathosense/3D-Quant-Surface-Visualization.git
   cd 3D-Quant-Surface-Visualization
   ```
3. **Install dependencies**:
   ```bash
   pip install numpy matplotlib scipy
   ```
4. **Run the visualization**:
   ```bash
   python main.py
   ```
